**Architecture:** Add a shared `gomoku.js` rule module, store one Gomoku state per room on the server, and render a new H5 tab that sends place/reset actions over the existing WebSocket connection.

**Tech Stack:** Node.js, node:test, HTML5, JavaScript, CSS, existing WebSocket room service.
